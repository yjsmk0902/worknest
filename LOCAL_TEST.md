## 특히 불안한 부분 (발견하면 바로 제보)

1. **BlockId 주입이 autosave 루프랑 충돌** — 타이핑 중 저장 스파이크 생기는지 네트워크 탭 확인
2. **History snapshot이 빈 문서 / 같은 내용에도 돌지** — "제목만 한 글자 바꿨는데 리비전 생김" 같은 과민 반응
3. **Toolbar가 새로 mount돼서 기존 UI 가리는 케이스** — 특히 위쪽 제목 근처에서 선택 시
4. **`/wiki-share/$token` 공개 페이지에 커스텀 CSS 변수(`--bg-0` 등)가 로드되는지** — `_app` 바깥이라 스타일이 안 먹힐 가능성
