# Worknest

**A-Team** | Jira + Confluence 대체 올인원 프로젝트 관리 플랫폼

> 이슈 트래킹, 스프린트 관리, Wiki 실시간 협업 편집을 하나의 셀프호스팅 가능한 앱으로.

## Status

프로젝트 기획 단계입니다. [기능정의서](docs/specs/FEATURE_SPEC.md)를 참고하세요.

## Tech Stack (예정)

| 영역 | 기술 |
|------|------|
| Frontend | Vite + React 19 + TypeScript + Tailwind CSS 4 |
| Backend | Node.js + Fastify + TypeScript |
| Database | PostgreSQL + Drizzle ORM |
| Real-time | WebSocket + Hocuspocus (Yjs CRDT) |
| Editor | TipTap (ProseMirror) |
| Infra | pnpm + Turborepo + Docker |
