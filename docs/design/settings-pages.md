# Worknest 설정 페이지 설계

> 이 문서는 워크스페이스 설정, 멤버 관리, 조직 설정, 프로젝트 설정 화면의 레이아웃과 컴포넌트 스펙을 정의합니다.
> 디자인 시스템 토큰은 `design-system.md`를 참조합니다.

---

## 1. 설정 페이지 공통 레이아웃

모든 설정 페이지는 좌측 탭 내비게이션 + 우측 콘텐츠 구조를 따릅니다.

```
┌── Sidebar ──┬── Settings Layout ────────────────────────────────┐
│             │                                                    │
│  (앱 셸     │  ┌─ Header ─────────────────────────────────────┐  │
│   사이드바)  │  │  ⚙ 워크스페이스 설정                           │  │
│             │  └───────────────────────────────────────────────┘  │
│             │                                                    │
│             │  ┌─ Settings Nav ─┬─ Settings Content ────────┐   │
│             │  │                │                            │   │
│             │  │  일반          │  (선택된 탭의 콘텐츠)        │   │
│             │  │  멤버          │                            │   │
│             │  │  ──────────   │  max-width: 720px          │   │
│             │  │               │  padding: p-6              │   │
│             │  │               │                            │   │
│             │  │               │                            │   │
│             │  │               │                            │   │
│             │  │  w-[200px]    │  flex-1                    │   │
│             │  │               │                            │   │
│             │  └───────────────┴────────────────────────────┘   │
│             │                                                    │
└─────────────┴────────────────────────────────────────────────────┘
```

### 1.1 Settings Nav (좌측 탭)

```
┌──────────────────┐
│                  │
│  일반              │  ← 활성: bg-accent font-medium rounded-md
│  멤버              │     비활성: text-muted-foreground
│                  │     hover: bg-accent/50
│                  │     h-8, px-3, text-sm
│                  │     w-[200px]
│                  │
└──────────────────┘
```

| 속성 | 값 |
|------|-----|
| 너비 | **200px** |
| 항목 높이 | `h-8` (32px) |
| 패딩 | `px-3` |
| 텍스트 | `text-sm` |
| 활성 | `bg-accent text-accent-foreground font-medium rounded-md` |
| 비활성 | `text-muted-foreground` |
| 호버 | `bg-accent/50 rounded-md` |
| 간격 | `space-y-1` |

### 1.2 Settings Content (우측 콘텐츠)

| 속성 | 값 |
|------|-----|
| 최대 너비 | **720px** |
| 패딩 | `p-6` |
| 섹션 간격 | `space-y-8` (32px) |
| 섹션 제목 | `text-lg font-semibold` + 설명 `text-sm text-muted-foreground` |
| 구분선 | `border-b border-border` (섹션 사이) |

## 2. 워크스페이스 설정

### 2.1 라우트

```
/:orgSlug/:wsSlug/settings          → 일반 (기본)
/:orgSlug/:wsSlug/settings/members  → 멤버
```

### 2.2 탭 구성

```
┌──────────────────┐
│  일반              │  ← /:orgSlug/:wsSlug/settings
│  멤버              │  ← /:orgSlug/:wsSlug/settings/members
└──────────────────┘
```

### 2.3 일반 설정 (General)

#### 기본 상태

```
┌───────────────────────────────────────────────────────────────┐
│                                                               │
│  워크스페이스 설정                                               │  ← text-lg font-semibold
│  워크스페이스의 기본 정보를 관리합니다.                            │  ← text-sm text-muted-foreground
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
│  로고                                                         │  ← text-sm font-medium
│  ┌──────────┐                                                │
│  │          │  ← 이미지 업로드 영역                              │
│  │  [아이콘]  │     64x64px, rounded-lg                         │
│  │          │     클릭 또는 드래그로 업로드                       │
│  └──────────┘                                                │
│  [변경] [제거]                                                  │  ← ghost 버튼, text-xs
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
│  워크스페이스 이름                                               │
│  ┌─────────────────────────────────────────┐                 │
│  │ 개발팀                                   │                 │
│  └─────────────────────────────────────────┘                 │
│                                                               │
│  워크스페이스 URL (slug)                                       │
│  ┌─────────────────────────────────────────┐                 │
│  │ dev-team                                │                  │  ← 수정 가능
│  └─────────────────────────────────────────┘                 │
│  worknest.app/acme-corp/dev-team                              │  ← 프리뷰
│                                                               │
│  설명                                                         │
│  ┌─────────────────────────────────────────┐                 │
│  │ 백엔드/프론트엔드 개발팀                    │                 │
│  │                                         │                 │
│  └─────────────────────────────────────────┘                 │
│                                                               │
│  ┌────────────────┐                                          │
│  │   변경사항 저장   │                                          │  ← primary 버튼
│  └────────────────┘                                          │     변경 시에만 활성화
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
│  위험 영역                                                     │  ← text-lg font-semibold
│                                                               │     text-destructive
│  ┌─────────────────────────────────────────────────────────┐ │
│  │                                                         │ │
│  │  워크스페이스 삭제                                        │ │  ← border border-destructive/20
│  │  이 워크스페이스와 모든 데이터가 삭제됩니다.                 │ │     rounded-lg p-4
│  │  이 작업은 30일 이내에 복구할 수 있습니다.                  │ │     bg-destructive/5
│  │                                                         │ │
│  │                              ┌──────────────────────┐   │ │
│  │                              │  워크스페이스 삭제      │   │ │  ← destructive 버튼
│  │                              └──────────────────────┘   │ │
│  │                                                         │ │
│  └─────────────────────────────────────────────────────────┘ │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

#### 로고 업로드 컴포넌트

| 상태 | 설명 |
|------|------|
| 기본 | 현재 로고 표시 (없으면 이니셜 아바타) |
| 호버 | 오버레이 + 카메라 아이콘 |
| 업로드 중 | 스피너 오버레이 |
| 에러 | `"이미지 업로드에 실패했습니다"` 토스트 |

| 속성 | 값 |
|------|-----|
| 크기 | 64x64px |
| 형태 | `rounded-lg` |
| 허용 파일 | `.jpg`, `.png`, `.webp` |
| 최대 크기 | 2MB |
| 표시 | 이니셜 폴백 (이름 첫 글자, `bg-primary text-primary-foreground`) |

#### "변경사항 저장" 버튼 동작

| 상태 | 표시 |
|------|------|
| 변경 없음 | `disabled`, `opacity-50` |
| 변경 있음 | 활성화 |
| 저장 중 | `[스피너] 저장 중...`, `disabled` |
| 저장 성공 | 토스트: `"설정이 저장되었습니다"` |
| 저장 실패 | 토스트: `"저장에 실패했습니다. 다시 시도해주세요."` |

#### 삭제 확인 모달

```
┌─────────────────────────────────────────────┐
│                                             │
│  워크스페이스 삭제                             │  ← text-lg font-semibold
│                                             │
│  정말 "개발팀" 워크스페이스를 삭제하시겠습니까?   │  ← text-sm
│  모든 프로젝트, 이슈, Wiki 페이지가              │
│  삭제됩니다. 30일 이내에 복구할 수 있습니다.     │
│                                             │
│  확인을 위해 워크스페이스 이름을 입력해주세요:    │
│  ┌──────────────────────────────────┐       │
│  │                                  │       │
│  └──────────────────────────────────┘       │
│                                             │
│  ┌───────────────┐  ┌──────────────────┐   │
│  │     취소       │  │  삭제             │   │  ← destructive, 이름 일치 시 활성화
│  └───────────────┘  └──────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 모달 크기 | 400px (소) |
| 삭제 버튼 | `variant="destructive"`, 이름 정확 일치 전 `disabled` |
| 입력 | 워크스페이스 이름 정확히 입력해야 활성화 |
| Esc | 닫기 |

## 3. 멤버 페이지 (Members)

### 3.1 기본 상태

```
┌───────────────────────────────────────────────────────────────┐
│                                                               │
│  멤버                                                         │  ← text-lg font-semibold
│  워크스페이스 멤버를 관리합니다.                                  │
│                                                               │
│  ┌─────────────────────────────────┐  ┌──────────────────┐   │
│  │ 🔍 멤버 검색...                  │  │  ＋ 멤버 초대      │   │  ← primary 버튼
│  └─────────────────────────────────┘  └──────────────────┘   │
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
│  ┌─ 멤버 테이블 ──────────────────────────────────────────┐   │
│  │                                                         │   │
│  │  이름               이메일              역할      작업    │   │  ← 테이블 헤더
│  │  ─────────────────────────────────────────────────────  │   │     text-xs font-medium
│  │                                                         │   │     text-muted-foreground
│  │  [아바타] Luke Kim   luke@acme.com    [Admin ▼]  [···]  │   │     uppercase tracking-wider
│  │  [아바타] Kim Park   kim@acme.com     [Member ▼] [···]  │   │
│  │  [아바타] Lee Min    lee@acme.com     [Guest ▼]  [···]  │   │  ← 행 높이: 56px
│  │                                                         │   │     hover: bg-accent/50
│  └─────────────────────────────────────────────────────────┘   │
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
│  대기 중인 초대                                                 │  ← 초대가 있을 때만 표시
│                                                               │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │                                                         │ │
│  │  이메일               역할       만료일      작업        │ │
│  │  ─────────────────────────────────────────────────────  │ │
│  │  new@acme.com       Member    4일 남음    [재발송] [취소]│ │
│  │  guest@other.com    Guest     6일 남음    [재발송] [취소]│ │
│  │                                                         │ │
│  └─────────────────────────────────────────────────────────┘ │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

### 3.2 멤버 테이블 컴포넌트

#### 테이블 헤더

| 컬럼 | 너비 | 정렬 |
|------|------|------|
| 이름 (아바타 + 이름) | flex-1 | 좌측 |
| 이메일 | 200px | 좌측 |
| 역할 | 120px | 좌측 |
| 작업 | 48px | 중앙 |

#### 테이블 행

```
┌────────────────────────────────────────────────────────────────┐
│  [아바타 32px]  Luke Kim        luke@acme.com    [Admin ▼] [···]│
│                 Admin                                          │
└────────────────────────────────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 행 높이 | 56px |
| 아바타 | 32px, `rounded-full` |
| 이름 | `text-sm font-medium` |
| 이메일 | `text-sm text-muted-foreground` |
| 호버 | `bg-accent/50` |
| 구분선 | `border-b border-border` (행 사이) |

#### 역할 드롭다운

```
클릭 시:
┌──────────────────┐
│  Admin            │  ← hover: bg-accent
│  ✓ Member         │  ← 현재 선택
│  Guest            │
└──────────────────┘
```

| 속성 | 값 |
|------|-----|
| 트리거 | `[역할명 ▼]`, `variant="outline"`, `text-xs` |
| 드롭다운 | Radix `Select` 또는 `DropdownMenu` |
| 항목 | `text-sm`, hover: `bg-accent` |
| 현재 선택 | `Check` 아이콘 |

**권한 규칙:**
- Admin만 다른 멤버의 역할 변경 가능
- 자기 자신의 역할 변경 불가
- Owner 역할은 표시만 (변경 불가)
- Admin이 아닌 멤버: 드롭다운 비활성 (`disabled`)

#### 작업 메뉴 (`···`)

```
클릭 시:
┌──────────────────┐
│  프로필 보기       │
│  ──────────────  │
│  멤버 제거        │  ← text-destructive
└──────────────────┘
```

**권한 규칙:**
- Admin만 "멤버 제거" 표시
- 자기 자신에게는 "워크스페이스 나가기"로 변경
- Owner는 제거 불가 (메뉴 항목 비활성)

### 3.3 멤버 초대 모달

```
┌─────────────────────────────────────────────┐
│                                             │
│  멤버 초대                                   │  ← text-lg font-semibold
│                                             │
│  이메일 주소                                  │
│  ┌──────────────────────────────────┐       │
│  │ email@company.com                │       │
│  └──────────────────────────────────┘       │
│                                             │
│  역할                                        │
│  ┌──────────────────────────────────┐       │
│  │ Member                        ▼  │       │  ← Select
│  └──────────────────────────────────┘       │
│                                             │
│  ┌───────────────┐  ┌──────────────────┐   │
│  │     취소       │  │   초대 보내기      │   │  ← primary 버튼
│  └───────────────┘  └──────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

| 상태 | 설명 |
|------|------|
| 기본 | 이메일 입력 + 역할 선택 |
| 로딩 | `[스피너] 초대 보내는 중...` |
| 성공 | 모달 닫힘 + 토스트: `"email@company.com에게 초대를 보냈습니다"` |
| 이미 멤버 | 에러: `"이미 워크스페이스 멤버입니다"` |
| 이미 초대됨 | 에러: `"이미 초대가 발송되었습니다"` |
| 이메일 형식 에러 | 필드 에러: `"올바른 이메일 주소를 입력해주세요"` |

#### 역할 선택 옵션

| 역할 | 설명 |
|------|------|
| Admin | 워크스페이스 전체 관리 권한 |
| Member | 프로젝트 참여, Wiki 편집 |
| Guest | 할당된 이슈와 허용된 스페이스만 접근 |

### 3.4 멤버 제거 확인 모달

```
┌─────────────────────────────────────────────┐
│                                             │
│  멤버 제거                                   │  ← text-lg font-semibold
│                                             │
│  "Kim Park" 님을 워크스페이스에서              │
│  제거하시겠습니까?                             │
│                                             │
│  이 멤버의 이슈 할당은 유지되지만,              │
│  워크스페이스에 더 이상 접근할 수 없습니다.      │
│                                             │
│  ┌───────────────┐  ┌──────────────────┐   │
│  │     취소       │  │     제거          │   │  ← destructive 버튼
│  └───────────────┘  └──────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

### 3.5 대기 중인 초대 테이블

| 컬럼 | 너비 |
|------|------|
| 이메일 | flex-1 |
| 역할 | 100px |
| 만료일 | 100px |
| 작업 | 120px |

```
│  new@acme.com       Member    4일 남음    [재발송] [취소] │
```

| 요소 | 스펙 |
|------|------|
| 재발송 | `ghost` 버튼, `text-xs` |
| 취소 | `ghost` 버튼, `text-xs text-destructive` |
| 만료일 | 3일 이하: `text-destructive`, 그 외: `text-muted-foreground` |

### 3.6 Empty State (멤버)

```
┌───────────────────────────────────────────┐
│                                           │
│           [Users 아이콘, 48px, muted]      │
│                                           │
│        아직 다른 멤버가 없습니다              │
│        팀원을 초대하여 함께 작업해보세요       │
│                                           │
│           ┌──────────────────┐            │
│           │   ＋ 멤버 초대     │            │
│           └──────────────────┘            │
│                                           │
└───────────────────────────────────────────┘
```

### 3.7 검색 동작

| 속성 | 값 |
|------|-----|
| 입력 필드 | `w-[280px]`, `placeholder="멤버 검색..."` |
| 검색 대상 | 이름, 이메일 |
| 검색 방식 | 클라이언트 사이드 필터 (즉시) |
| 결과 없음 | `"검색 결과가 없습니다"` 인라인 메시지 |

## 4. 조직 설정 (Organization Settings)

### 4.1 접근 경로

사이드바 하단 사용자 메뉴 → "조직 설정" 또는 Org/WS 선택기 → "조직 설정"

### 4.2 탭 구성

```
┌──────────────────┐
│  일반              │  ← 조직 기본 정보
│  멤버              │  ← 조직 멤버 관리
│  워크스페이스       │  ← 워크스페이스 목록
└──────────────────┘
```

### 4.3 일반 설정

```
┌───────────────────────────────────────────────────────────────┐
│                                                               │
│  조직 설정                                                     │
│  조직의 기본 정보를 관리합니다.                                  │
│                                                               │
│  ─────────────────────────────────────────────────────────── │
│                                                               │
```
