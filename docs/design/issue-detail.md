# Worknest 이슈 상세 화면 설계 (CP2-DS-2)

> 이 문서는 이슈 상세 화면의 Side Panel과 Full Page 레이아웃, 속성 사이드바, 에디터, 서브이슈, 활동 로그의 컴포넌트 스펙을 정의합니다.
> 디자인 시스템 토큰은 `design-system.md`를 참조합니다.

---

## 1. 레이아웃 모드

이슈 상세는 두 가지 레이아웃을 지원합니다.

| 모드 | 너비 | 사용 조건 |
|------|------|----------|
| **Side Panel** | 640px | 이슈 리스트/보드에서 클릭 시 (1024px+ 화면) |
| **Full Page** | 전체 콘텐츠 영역 | 이슈 키 직접 접근, Side Panel에서 전체화면 전환, 1024px 미만 |

### 1.1 라우트

```
Side Panel:  /:orgSlug/:wsSlug/projects/:prefix/issues (이슈 리스트 유지)
Full Page:   /:orgSlug/:wsSlug/projects/:prefix/issues/:issueKey
```

## 4. 속성 컴포넌트 스펙

### 4.1 상태 드롭다운

```
트리거:
┌─────────────────────────┐
│  [●] In Progress     ▼  │  ← 색상 dot (8px) + 상태명
└─────────────────────────┘     border border-border rounded-md
                                h-8, px-2, text-sm
                                hover: bg-accent
드롭다운:
┌─────────────────────────────┐
│  [●] Backlog                │  ← 카테고리별 색상 dot
│  [●] Todo                   │     hover: bg-accent
│  ─────────────────────────  │
│  [●] In Progress      ✓    │  ← 현재 선택
│  [●] In Review              │
│  ─────────────────────────  │
│  [●] Done                   │
│  [●] Cancelled              │
└─────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 컴포넌트 | Radix `Select` |
| dot | 8px `rounded-full`, 카테고리 색상 (`--status-{category}-text`) |
| 그룹핑 | 카테고리 구분선으로 그룹 분리 (Backlog/Unstarted, Started, Completed/Cancelled) |
| 현재 선택 | `Check` 아이콘 (Lucide, 16px) |
| 변경 시 | 낙관적 업데이트 + API 호출 |
| 단축키 | `S` (이슈 상세 컨텍스트) |

### 4.2 우선순위 드롭다운

```
트리거:
┌─────────────────────────┐
│  [⚡] High            ▼  │  ← 우선순위 아이콘 + 이름
└─────────────────────────┘

드롭다운:
┌─────────────────────────────┐
│  [‼️] Urgent                │  ← 빨강 아이콘
│  [⚡] High            ✓     │  ← 주황 아이콘, 현재 선택
│  [━] Medium                 │  ← 노랑 아이콘
│  [↓] Low                    │  ← 파랑 아이콘
│  [─] None                   │  ← 회색 아이콘
└─────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 컴포넌트 | Radix `Select` |
| 아이콘 | 우선순위별 고유 아이콘, 크기 16px |
| 아이콘 색상 | `--priority-{level}` 색상 |
| 변경 시 | 낙관적 업데이트 + API 호출 |
| 단축키 | `P` (이슈 상세 컨텍스트) |

#### 우선순위 아이콘 매핑

| 우선순위 | 아이콘 | Lucide 이름 |
|---------|--------|-------------|
| Urgent | 이중 느낌표 | `AlertTriangle` |
| High | 번개 | `Zap` |
| Medium | 가로선 | `Minus` |
| Low | 아래 화살표 | `ArrowDown` |
| None | 대시 | `Minus` (회색) |

| 오늘 | `border border-primary` (선택 전) |
| 과거 날짜 | 선택 가능 (비활성화 없음) |
| 날짜 제거 | 마감일 제거 (`ghost` 버튼, `text-destructive`) |
| 오늘 버튼 | 오늘 날짜로 즉시 설정 |
| 변경 시 | 낙관적 업데이트 + API 호출 |
| 단축키 | `D` (이슈 상세 컨텍스트) |
| 트리거 표시 | 날짜 설정 시: `"YYYY-MM-DD"`, 미설정: `"마감일 없음"` (`text-muted-foreground`) |

#### 마감일 표시 규칙

| 조건 | 표시 | 색상 |
|------|------|------|
| 기한 남음 (3일+) | `YYYY-MM-DD` | `text-foreground` |
| 기한 임박 (1~3일) | `YYYY-MM-DD` | `text-orange-500` |
| 오늘 | `오늘 마감` | `text-orange-500 font-medium` |
| 기한 초과 | `N일 초과` | `text-destructive font-medium` |
| 완료 상태 | ~~`YYYY-MM-DD`~~ | `text-muted-foreground line-through` |

## 11. 접근성 (Accessibility)

### 11.1 포커스 관리

- Side Panel 열림: Panel 내부 첫 번째 요소로 포커스 이동
- Side Panel 닫힘: 트리거 요소(이슈 리스트 행)로 포커스 복귀
- 제목 편집: `F2` 또는 클릭으로 편집 모드 진입, `Esc` 시 읽기 모드 복귀 및 제목에 포커스 유지

### 11.2 ARIA

- Side Panel: `role="complementary" aria-label="이슈 상세"`
- 제목 (읽기): `role="heading" aria-level="1"`
- 제목 (편집): `aria-label="이슈 제목 편집"`
- 속성 드롭다운: `aria-label="{속성명} 변경"`
- 담당자/라벨 Popover: `aria-label="{속성명} 선택"`, `aria-expanded`
- 서브이슈 체크박스: `aria-label="{이슈 키} - {제목}"`
- 활동 로그: `role="feed" aria-label="활동 로그"`
